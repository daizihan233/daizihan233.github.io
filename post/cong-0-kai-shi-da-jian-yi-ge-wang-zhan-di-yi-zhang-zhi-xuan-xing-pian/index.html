开头

读了这篇文章您将学到

  1. 各个托管平台之间的区别
  2. 什么是动（静）态网页
  3. 静态网页生成器的选择

平台的选择

现在最受欢迎的有以下几个平台

  1. GitHub Pages
  2. Coding Pages（已弃用）
  3. Gitee Pages
  4. 自己动手，丰衣足食（自己搭建）

如何选择？

总结：建议选择GitHub Pages

GitHub Pages

非常牛逼的静态网页托管平台
在未满18岁（无法备案）的前提下很好用，不用备案，自带域名（可自定义）**，但国内连接速度很慢

小知识：什么事静态/动态网页？

静态网页

比如个人博客，比如你现在看的这个网站

动态网页

比如哔哩哔哩，所有用户都可以上传作品

Coding Pages（已弃用）

可大大提高国内的速度，但是收费的，看看得了
别看了，已经关停了，沙*tx（危险发言）

Gitee Pages

需要备案，需要身份证正反面以及手持身份证照片，国内速度更快

自己动手，丰衣足食（自己搭建）

需要注意的是，前面几个只支持静态网页，不支持动态网页，自己动手当然是个性化程度最高的方式，但也麻烦，你也不想自己一行一行写代码对吧，我也不想

静态网页生成器

建议：Gridea
目前我见过最好的是Gridea，你没看错，确实不是WordPress
WordPress需要数据库的支持，所以在这里我们不用他
Gridea主要是做博客的，有模板可以直接套，还可以直接上传到GitHub
我保证没有恰饭

结尾

后面会用GitHub Pages+Gridea从无到有制作一个网站
